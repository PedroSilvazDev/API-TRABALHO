# Atividades de Machine Learning

Notebooks prontos para **Google Colab** — abra o arquivo `.ipynb` no Colab e execute todas as células em ordem.

| Arquivo | Tema | Algoritmo |
|---------|------|-----------|
| `01-RNA-Perceptron-MLP.ipynb` | Perfis de clientes de delivery | Perceptron e MLP |
| `02-Apriori-Associacao.ipynb` | Pedidos em loja online de moda | Apriori |
| `03-Agrupamento-Clustering.ipynb` | Assinantes de streaming | K-Means, AGNES, DBSCAN |
| `04-SVM-Classificacao.ipynb` | Símbolos gráficos (estrela, cruz, losango) | SVM |

## Como usar no Colab

1. Acesse [colab.research.google.com](https://colab.research.google.com)
2. **Arquivo → Fazer upload do notebook** (ou abra direto do GitHub)
3. **Runtime → Executar tudo**

Nenhum arquivo externo é necessário — todos os dados são gerados dentro do notebook.
